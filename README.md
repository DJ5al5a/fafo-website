# Friends and Family Outreach (FAFO) Website

A comprehensive resource website for families dealing with Florida's child welfare system.

## About This Website

Friends and Family Outreach (FAFO) is a community-led advocacy and resource organization providing educational information and support for families involved with the Department of Children and Families (DCF) in Florida.

**Important:** FAFO is NOT a law firm and does not provide legal advice.

## Website Structure

This website contains the following pages:

### Main Pages

1. **index.html** - Home page
   - Introduction to FAFO
   - Mission and purpose
   - Quick links to all resources

2. **know-your-rights.html** - Know Your Rights as a Parent
   - Constitutional rights (4th, 5th, 14th Amendments)
   - Rights during DCF contact
   - Rights during investigations
   - Common misunderstandings

3. **dcf-policies.html** - DCF Policies & Procedures
   - What DCF is legally required to do
   - Investigation process and timelines
   - Required procedures
   - Common procedural failures

4. **state-laws.html** - Florida State Laws for Children & Families
   - Chapter 39 Florida Statutes explained
   - Key statute summaries
   - Dependency process timeline
   - Links to official sources

5. **reporting.html** - Reporting & Accountability (HIGH PRIORITY)
   - How to file complaints against DCF
   - Florida complaint pathways (DCF OIG, Governor's Office)
   - Federal complaint pathways (HHS, DOJ, FBI)
   - Documentation tips

6. **resources.html** - Support Resources
   - Legal aid organizations (statewide and regional)
   - Court-appointed attorney information
   - Advocacy organizations
   - Emergency resources

7. **contact.html** - Contact Us
   - Contact form
   - Information about how FAFO can help
   - Clear disclaimers

8. **legal-disclaimer.html** - Legal Disclaimer
   - Full legal disclaimer
   - Explanation that FAFO is not a law firm
   - Important notices

### Supporting Files

- **styles.css** - Complete website styling
- **script.js** - Navigation, accordion, and form functionality
- **README.md** - This file

## How to Use This Website

### Viewing Locally

1. Open `index.html` in any web browser
2. All navigation links will work locally
3. The contact form is client-side only (you'll need to set up backend processing)

### Deploying to a Web Server

1. Upload all files to your web hosting service
2. Ensure all files maintain their structure in the same directory
3. Set up form submission handler for contact.html (currently shows alert only)
4. Point your domain to the hosting location

## Contact Form Setup

The contact form currently uses JavaScript to show an alert. To make it functional:

1. **Option 1: Email Service**
   - Use a service like Formspree, EmailJS, or SendGrid
   - Update form action in contact.html

2. **Option 2: Backend Server**
   - Set up PHP, Node.js, or Python backend
   - Configure email sending
   - Update form submission handler

3. **Option 3: Static Form Service**
   - Use Netlify Forms, FormSubmit, or similar
   - Follow service documentation
